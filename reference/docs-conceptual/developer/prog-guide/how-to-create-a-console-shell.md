<span data-ttu-id="afd81-103">Windows PowerShell은 확장 가능 하지 않은 콘솔 셸을 만드는 데 사용 되는 셸 만들기 도구 ("키트" 라고도 함)를 제공 합니다.</span><span class="sxs-lookup"><span data-stu-id="afd81-103">Windows PowerShell provides a Make-Shell tool, also referred to as the "make-kit", that is used to create a console shell that is not extensible.</span></span> <span data-ttu-id="afd81-104">이 새로운 도구를 사용 하 여 만든 셸을 나중에 Windows PowerShell 스냅인을 통해 확장할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="afd81-104">Shells created with this new tool cannot be extended later through a Windows PowerShell snap-in.</span></span>

## <a name="syntax"></a><span data-ttu-id="afd81-105">구문</span><span class="sxs-lookup"><span data-stu-id="afd81-105">Syntax</span></span>

<span data-ttu-id="afd81-106">다음은 파일에서 만들기-셸을 실행 하는 데 사용 되는 구문입니다.</span><span class="sxs-lookup"><span data-stu-id="afd81-106">Here is the syntax used to run Make-Shell from within a make-file.</span></span>

```
make-shell
  -out n.exe
  -namespace ns
  [ -lib libdirectory1[,libdirectory2,..] ]
  [ -reference ca1.dll[,ca2.dll,...] ]
  [ -formatdata fd1.format.ps1xml[,fd2.format.ps1xml,...] ]
  [ -typedata td1.type.ps1xml[,td2.type.ps1xml,...] ]
  [ -source c1.cs [,c2.cs,...] ]
  [ -authorizationmanager authorizationManagerType ]
  [ -win32icon i.ico ]
  [ -initscript p.ps1 ]
  [ -builtinscript s1.ps1[,s2.ps1,...] ]
  [ -resource resourcefile.txt ]
  [ -cscflags cscFlags ]
  [ -? | -help ]
```
