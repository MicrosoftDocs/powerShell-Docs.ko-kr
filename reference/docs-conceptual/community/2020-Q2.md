# <a name="whats-new-in-powershell-docs-for-2020-q2"></a>PowerShell Docs의 새로운 기능 2020년 2분기

이 문서에서는 이전 달에 발생한 문서에 대한 주요 변경 내용을 나열하고 커뮤니티 기여에 대한 감사를 전합니다.

설명서를 개선하는 데 도움을 주세요. 시작하는 방법을 알아보려면 [기여자 가이드][contrib]를 읽으세요.

## <a name="2020-june-highlights"></a>2020년 6월 주요 사항

- 새로운 PowerShell 설명서
  - Mike F. Robbins가 제공한 새 [PowerShell 101](../learn/ps101/00-introduction.md) 콘텐츠를 게시함
  - Rob Holt의 최근 블로그 게시물 두 개를 **스크립팅 및 개발** 문서에 추가함
    - [.NET 프로젝트에 적합한 PowerShell NuGet 패키지 선택](/powershell/scripting/dev-cross-plat/choosing-the-right-nuget-package)
    - [PowerShell 모듈 어셈블리 종속성 충돌 해결](/powershell/scripting/dev-cross-plat/resolving-dependency-conflicts)
- 설명서 유지 관리
  - 이전 콘텐츠를 [https://aka.ms/PSLegacyDocs](https://aka.ms/PSLegacyDocs)에 보관
    - PowerShell 웹 액세스 SDK 콘텐츠
    - PowerShell 워크플로 SDK 콘텐츠
  - 7\.1-미리 보기 콘텐츠 업데이트

  - 83개 PR 병합(커뮤니티에서 15개)
