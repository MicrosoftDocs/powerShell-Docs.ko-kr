# <a name="whats-new-in-powershell-docs"></a>PowerShell Docs의 새로운 기능

이 문서에서는 이전 달에 발생한 문서에 대한 주요 변경 내용을 나열하고 커뮤니티 기여에 대한 감사를 전합니다.

설명서를 개선하는 데 도움을 주세요. 시작하는 방법을 알아보려면 [기여자 가이드][contrib]를 읽으세요.
