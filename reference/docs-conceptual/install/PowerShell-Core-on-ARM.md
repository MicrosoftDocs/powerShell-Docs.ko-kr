# <a name="powershell-core-on-arm"></a>ARM에서 PowerShell Core

현재 PowerShell Core는 Windows 10 ARM32/ARM64 및 Raspbian을 비롯한 일부 ARM 디바이스에서 작동합니다.
ARM용 PowerShell Core는 지원되지 않는 실험적 릴리스입니다.

이러한 실험적 빌드를 설치하는 방법에 대한 자세한 내용은 [Windows 10 IoT Enterprise](installing-powershell-core-on-windows.md#deploying-on-windows-10-iot-enterprise),[Windows 10 IoT Core](installing-powershell-core-on-windows.md#deploying-on-windows-10-iot-core) 및 [Raspbian](installing-powershell-core-on-linux.md#raspbian)에 대한 설치 지침을 참조하세요.

ARM64의 진행 현황에 대한 자세한 정보를 여기서 볼 수 있습니다.

기대해 주세요.
